Models utilities

Functions​

fill​

Fill the model instance's values with the given values.

Example​

import { fill } from 'func-client/core';

const post = fill(new Post(), { title: 'Hello', description: 'World' });

Arguments​

  • {I extends ModelInstance} instance
  • {Partial<ModelValues<I>>} values

Returns​

{I}, the affected instance.

changed​

Check if the given keys have been changed since last instance sync. If no keys are provided, will check all instance keys.

Example​

import { changed } from 'func-client/core';

const wasChanged = changed(post, ['title', 'description']);
if (wasChanged) { /* do something */ }

Arguments​

  • {I extends ModelInstance} instance
  • {ArrayableVariadic<ModelKey<I>} ...keys

Returns​

{boolean}

syncOriginal​

Sync the model instance's values as it was the original data source values. If no keys are provided, will sync all instance keys.

Example​

import { syncOriginal } from 'func-client/core';

syncOriginal(post, ['title', 'description']);

Arguments​

  • {I extends ModelInstance} instance
  • {ArrayableVariadic<ModelKey<I>>} ...keys

Returns​

{I}, the affected instance.

reset​

Reset the model instance's original values. If no keys are provided, will reset all instance keys.

Example​

import { reset } from 'func-client/core';

reset(post, ['title', 'description']);

Arguments​

  • {I extends ModelInstance} instance
  • {ArrayableVariadic<ModelKey<I>>} ...keys

Returns​

{I}, the affected instance.

loaded​

Check if the given relations are loaded on the instance or its related instances. Can check for sub relations using dot relation keys: will check each related instances regardless of the number of concerned instances. If no relations are provided, wont perform any check.

Example​

import { loaded } from 'func-client/core';

const isFullyLoaded = loaded(post, ['comments', 'comments.author']);
if (!isFullyLoaded) { /* do something */ }

Arguments​

  • {I extends ModelInstance} instance
  • {ArrayableVariadic<ModelRelationDotKey<I>>} ...relations

Returns​

{boolean}

isSame​

Check if two values are the same model instance. Ensure the equality by checking if the two instances:

  • Are FuncClient model instances
  • Have the same type
  • Have the same and non-NIL ID

Example​

import { isSame } from 'func-client/core';

const areSameInstances = isSame(foo, bar);
if (areSameInstances) { /* do something */ }

Arguments​

  • {unknown} value
  • {unknown} otherValue

Returns​

{boolean}

makeModel​

Create a model class.

Example​

import { makeModel } from 'func-client/core';

const PostModel = makeModel('posts', { /* definition */ });

Arguments​

  • {ModelConfig | string} config
  • {ModelDefinition} rawDefinition

Returns​

{ModelClass}

makeModelFactory​

Create a model class factory.

Example​

import { makeModelFactory } from 'func-client/core';

const makeModel = makeModelFactory({ /* definition */ }, { /* config */ });

Arguments​

  • {ModelDefinition | undefined } baseRawDefinition
  • {ModelConfig | undefined} baseConfig (omitting type)

Returns​

{makeModel}, a customized model factory function.

makeComposable​

Create a composable definition to integrate in models.

Example​

import { makeComposable } from 'func-client/core';

const publishable = makeComposable({ /* definition */ });

Arguments​

  • {ModelDefinition} rawDefinition

Returns​

{ModelParsedDefinition}
